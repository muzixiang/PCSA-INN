# PCSA-INN

This repository contains the anonymized code for the paper 'PCSA-INN: Physics-Informed Conditional Spectral Attention Invertible Neural Network for Atmosphere-Water Decoupling in Multispectral Satellite Imagery' submitted to ACM MM 2026. To comply with the double-blind review process, author information has been removed. Due to confidentiality reasons, a functional subset of the core modules is provided for review, with the full implementation and pre-trained weights to be released upon formal publication.
